## What This Skill Does

Drafts a short, professional WhatsApp follow-up message tailored to where the client or lead is in the relationship. No emojis. Direct and human.

## Steps

### 1. Identify the situation

Ask the user (in one message):

- Who is this for? (client name or description, e.g. "Canada client", "new lead from Instagram")
- What type of follow-up?
  1. No response -- they went quiet after a quote or proposal
  2. Project update -- checking in mid-project or sharing progress
  3. After delivery -- post-launch check-in or asking for feedback
  4. Upsell -- offering an additional service to an existing client
  5. Cold lead re-engagement -- someone who showed interest but never converted
  6. Other (describe it)
- Any specific detail to include? (e.g. what was sent, how long ago, what's the current status)

If the user already provided context when invoking (e.g. `/follow-up Canada client hasn't replied in 3 days`), use it and only ask for what's missing.

---

### 2. Draft the follow-up message

Write a WhatsApp-ready message. Rules:
- 3-5 sentences max
- No emojis
- No em dashes
- Starts with a greeting using the client's name or "Hi" if unknown
- Sounds like a real person, not a template
- Ends with a clear, low-pressure action (question, invitation, or next step)

Tailor tone and content by type:

**Type 1 -- No response (went quiet)**
- Acknowledge the previous message briefly
- Restate one key value or point without being pushy
- Give them an easy out ("Happy to adjust if something didn't fit.")
- End with a simple question

**Type 2 -- Project update**
- State what has been completed or what stage you're at
- Note what comes next
- Ask if they have any questions or input needed

**Type 3 -- After delivery**
- Confirm the project is live or delivered
- Ask if everything looks good
- Offer the revision window reminder (e.g. "You have 2 weeks for minor tweaks.")
- Optional: ask for a testimonial or referral (only if the relationship feels warm)

**Type 4 -- Upsell**
- Acknowledge the current or completed project positively
- Introduce the new service naturally, tied to their situation
- Keep it a suggestion, not a pitch
- End with a question

**Type 5 -- Cold lead re-engagement**
- Reference the previous interaction briefly (no pressure)
- Share something new or relevant (a project you completed, a relevant service)
- End with a low-friction question

---

### 3. Offer a variation

After presenting the message, ask:
"Want a shorter version, a different tone, or a version in Bahasa Indonesia?"

---

## Rules

- No emojis.
- No em dashes.
- Keep it under 5 sentences -- WhatsApp messages should feel natural, not like emails.
- Never sound desperate or overly apologetic.
- If the user provides the client's name, use it in the greeting.
- If asked for Bahasa Indonesia, rewrite naturally -- do not translate word-for-word.
- Do not invent details about the project -- use only what the user tells you.
